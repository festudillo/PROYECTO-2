El proyecto consiste en preparar un informe para la división de préstamos de un banco. En él se verifica si el estado civil y el número de hijos de un cliente tienen un impacto en el incumplimiento de pago de un préstamo. El banco ya tiene algunos datos sobre la solvencia crediticia de los clientes. El proyecto se enfoca en crear una puntuación de crédito para un cliente potencial y para evaluar la capacidad de nuevos clientes para pagar su préstamo.
